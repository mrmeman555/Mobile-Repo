## Core Mechanism (Precisely)

Chosen belief functions as a weighting signal that determines which contextual information is treated as salient during memory reactivation and reconsolidation.

More specifically:

- When a memory is reactivated, it becomes labile (temporarily editable).
- During this window, the brain evaluates contextual inputs to decide how the memory should be re-encoded.
- Beliefs about self, agency, responsibility, and meaning determine:
  - what contextual information is admitted into working memory
  - how heavily that information is weighted
  - whether shame/threat signals dominate or are overridden

Your key insight was:

> The memory is not updated by “new facts” alone —  
> it is updated by which facts are allowed to matter.

And chosen belief is the mechanism that assigns that “matter.”

## Why belief matters more than facts

Two people can recall the same memory with the same additional facts and get opposite outcomes, because:

One holds a belief like:
“This reflects something fundamentally wrong with me.”

The other holds:
“This occurred under unfair conditions and does not define me.”

Those beliefs change the weighting, not the availability, of context.

So reconsolidation depends on:

- What context is present
- What context is trusted
- What context is allowed to update identity

That third point is critical.

## Your specific contribution to the model

You identified that:

- Shame acts as a global veto signal
- When shame dominates, contextual updates are blocked or distorted
- Chosen belief can hold context online despite shame
- Defiance (fierce self-compassion) prevents collapse into threat processing
- Loving presence allows reconsolidation without activating danger circuitry

So the full loop looks like:

1. Memory activates
2. Shame/threat signal fires
3. Chosen belief keeps alternative context in working memory
4. Threat response is prevented from taking control
5. Memory is re-encoded with new self-referential meaning
