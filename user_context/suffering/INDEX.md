# Index (suffering)

This folder stores suffering-related artifacts as:

- **Long-form artifacts (individual files)**: `YYYYMMDD_HHMMSS__<short-slug>.md`
- **Append-only index**: `suffering_log.md`

`suffering_log.md` is the append-only, chronological index of the long-form suffering artifacts.

This file exists only to explain the storage model, not to summarize contents.
